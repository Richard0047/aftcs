# 2. Peska 流程 (Peska / ペスカ)

> Peska 为 AFT 支持的本地/区域支付方式。AFT 账户与 Peska 的登记信息（姓名罗马字、邮箱、电话）必须**完全一致**，否则将出现 PSP 异常等错误。

---

## 2.1 操作步骤

| 步骤 | 动作 | 备注 |
|------|------|------|
| 1 | 关联既有交易账户 | 管理画面右上角「既存取引口座を連携」；或「取引口座」→「既存口座を紐付ける」 |
| 2 | 确认信息一致 | AFT 与 Peska 的姓名（罗马字）、邮箱、电话**必须完全一致** |
| 3 | 选择 Peska 入金 | 入金页选择 Peska，按订单号与金额在 Peska 端完成支付 |
| 4 | **个人资料修改** | 我的资料 → 设置/注册信息（详见下方） |
| 5 | 密码变更 | マイページ → 設定 → 設定 → パスワードを変更 |

### 个人资料修改详细步骤（姓名·电话·邮箱）

**邮箱修改**

| 顺序 | 动作 |
|------|------|
| 1 | 登录会员后台：https://portal.asiafuturetrading.com/login |
| 2 | 进入 **我的资料** |
| 3 | 点击 **设置** |
| 4 | 点击 **更新电子邮箱**，按提示完成修改 |

**姓名修改**

| 顺序 | 动作 |
|------|------|
| 1 | 登录会员后台：https://portal.asiafuturetrading.com/login |
| 2 | 进入 **我的资料** |
| 3 | 点击 **注册信息** |
| 4 | 点击右上角 **编辑按钮** |
| 5 | 修改 **氏名（Name）** 为**英文字母/罗马字**（例：TARO TANAKA）。不可使用汉字；姓名字间勿留多余空格 |
| 6 | 点击 **保存** |

**日文路径**：個人情報 → 新規登録情報 → 氏名（Name）→ アルファベット表記に変更 → 保存

> ※ 修改后将自动反映至支付要求，无需额外申请或审核，即可使用 Peska 支付。

---

## 2.2 安全原则

| 要点 | 说明 |
|------|------|
| **信息一致** | AFT 与 Peska 的姓名（罗马字）、邮箱、电话必须**完全一致**，否则入金/出金无法正常处理 |
| **姓名格式** | 姓名须为**英文字母/罗马字**（如 TARO TANAKA）。汉字或姓名字间多余空格会导致 PSP 异常 |
| **错误排查** | 出现「PSP異常」时，请客户点击右侧 ! 图标获取错误详情，便于技术部门排查 |

> [!CAUTION]
> 未经核对即承诺「马上到账」或代为修改客户登记信息，可能导致误操作与合规风险。信息修正须引导客户在 Portal 自助完成。
