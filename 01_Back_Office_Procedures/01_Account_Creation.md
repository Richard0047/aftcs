# 1. KYC 账户创建 (Account Creation / 口座開設)

---

## 1.1 前置校验

| 步骤 | 动作 | 备注 |
|------|------|------|
| 1 | 核对身份证明 (KYC / 本人確認) 与申请信息一致 | 姓名、证件号、国籍 |
| 2 | 确认邮箱、手机未与既有账户重复绑定 | 系统内检索后再创建 |
| 3 | 完成反洗钱 (AML / マネーロンダリング対策) 基础问询 | 按清单逐项确认 |

## 1.2 话术建议

| 类型 | 推荐用语 | 禁忌用语 |
|------|----------|----------|
| 创建前 | 「请您确认以下信息与证件一致，我们将为您开通账户。」 | 「随便填就行」「不用核对」 |
| 创建后 | 「账户已开通，登录链接与临时密码已发送至您预留邮箱，请 24 小时内修改密码。」 | 「密码我口头告诉您」 |

## 1.3 场景一：KYC 审核未通过 (KYC Rejection / KYC却下)

[请在此处填写原因，例如：证件照片不清晰 / 地址证明已超过3个月期限 / 证件边缘裁剪不完整]

为了能尽快为您开通账户，麻烦请您重新登录会员后台，参照要求重新上传清晰的文件。
如果您在操作中遇到任何困难，请随时联系我们。
```

## 1.5 场景三：KYC 报错已解决（人工审核完成、可继续办理）(KYC Resolved / KYC問題解消)

```

关于身份认证（KYC）报错一事，给您带来了不便，深表歉意。

针对此事，刚才已由专人完成了人工审核。
特此告知：通过快速通道（Fast Track）优先处理，目前已顺利通过认证。
系统上的重复报错也已解除，您现在可以登录个人中心继续办理后续手续。

此次给您添麻烦了，非常抱歉。
如果您有任何不明之处，或在后续步骤（如入金方法等）中遇到困难，请随时回复本邮件联系我们。
请多多关照。
```

## 1.6 场景四：KYC 已人工通过（无需重新上传）(KYC Manually Approved / KYC手動承認)

**适用**：客户曾因系统报错等在提交身份证明时卡住；后台已人工完成 KYC 审核。**告知结果 → 解决痛点（不需再上传）→ 行动指引（登录后台确认）。**

```

因此，**您无需再次上传**（如驾照等）证件。您的账户现已完全激活，所有功能均可使用。请登录会员后台确认您的状态。

如有任何疑问或需要协助，请随时联系我们。
此致，
AFT 客服团队
